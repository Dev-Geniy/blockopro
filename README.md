**Blocko Pro** — это не просто канбан-доска. Это высокопроизводительная экосистема для управления проектами в реальном времени, созданная для команд, которые ценят эстетику, скорость и мощь искусственного интеллекта. 

---

## 💎 Почему Blocko Pro уникален?

В отличие от перегруженных Jira или ClickUp, Blocko Pro фокусируется на **"потоковом" (Flow) опыте**:

* **⚡ Real-Time Engine:** Благодаря интеграции с Firebase Firestore, каждое перемещение карточки, каждое сообщение в чате и каждое изменение тега синхронизируется мгновенно у всей команды без перезагрузки страниц.
* **🤖 AI Project Manager:** Встроенный интеллект (OpenAI/Pollinations) не просто пишет текст, он генерирует структуру задач по методологии SMART, превращая краткую идею в пошаговый план реализации прямо внутри карточки.
* **🎨 Премиальный UX/UI:** Интерфейс построен на принципах "Glassmorphism" и "Dark Mode First". Использование шрифта Plus Jakarta Sans и плавных анимаций SortableJS создает ощущение работы с нативным приложением.
* **🔐 Прозрачная безопасность:** Мы используем Google Auth для мгновенного входа и кастомные Firestore Security Rules для защиты интеллектуальной собственности вашей компании.

## 🚀 Ключевой функционал

### 1. Управление проектами (The Grid)
* **Интеллектуальный поиск:** Фильтрация по названию и тегам с динамической подсветкой совпадений.
* **Система архивации:** Умное хранилище "Archive", позволяющее освобождать доску от завершенных дел, сохраняя историю для аудита.
* **Командный центр:** Возможность назначать роли (Владелец, Редактор, Читатель) и управлять доступом через инвайт-ссылки.

### 2. Карточки задач (The Atom)
* **Rich Content:** Поддержка Markdown, чек-листов с прогресс-барами и обложек через API ImgBB.
* **Дедлайны:** Визуальная индикация просроченных задач с пульсирующими уведомлениями.

### 3. Коммуникации (The Hub)
* **Проектный чат:** Отдельная ветка обсуждения для каждой доски, устраняющая необходимость в сторонних мессенджерах.

## 🛠 Технологический стек

* **Frontend:** HTML5, JavaScript (ES6+), TailwindCSS (JIT mode).
* **Backend-as-a-Service:** Firebase (Auth, Firestore, Hosting).
* **Libraries:** Lucide Icons (графика), SortableJS (драг-н-дроп), Marked.js (парсинг контента).
* **Media:** ImgBB API (хранение визуальных ассетов).

* **Blocko Ecosystem:** Полноценная замена Google Workspace / Microsoft 365, сфокусированная на креативных индустриях.

## ⚙️ Быстрый старт

1.  **Клонирование:** `git clone https://github.com/dev-geniy/blocko.git`
2.  **Настройка:** Внесите ваши ключи в `config.js`.
3.  **Безопасность:** Установите Firestore Rules из документации проекта.
4.  **Запуск:** Просто откройте `index.html` через Live Server.
